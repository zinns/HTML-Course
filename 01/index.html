¿Qué es la terminal? Se utiliza una terminal para poder interactuar con un ordenador a través de comandos o líneas de texto y poder hacer diversas funciones. ¿Cuáles son los comandos más utilizados? mkdir, touch, ls, pwd, open, mv, code .., command+s, entre otros. Ejemplos y descripción de comandos: mkdir - para crear una carpeta touch - para crear un archivo ls - para ver lo que hay dentro de una carpeta pwd - para mostrar la ruta open - para abrir finder mv - para mover archivos code .. - abre visual studio command + s - para guardar cambios
