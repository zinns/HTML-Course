¿Qué es git? Es un manejador o controlador de versiones. ¿Para qué sirve? Para comparar los cambios que se han hecho con versiones anteriores. ¿Cuáles son los comandos más utilizados? git add, git commit -m, git branch -v, git push -u origin, git init, git fetch, entre otros... Poner ejemplo y descripción de los comandos. git add . - para agregar los archivos que han tenido cambios git commit -m "___" - para agregar un commit git branch -v muestra el nombre de la rama y el último commit git push -u origin "nombre de la rama" - para enviar o subir cambios a github git init - para iniciar un repositorio en git git fetch - actualiza si hubo una nueva rama o versión de github
