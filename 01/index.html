HTML sus siglas significan (Hyper Text Markup Language) Principalmente utiliza "marcas" para etiquetar. Puedes usar etiquetas para darle propiedades al texto. Existen 5 versiones o estándares. HTML5 es la versión más actual. Ejemplos y descripción de comandos: mkdir - para crear una carpeta touch - para crear un archivo ls - para ver lo que hay dentro de una carpeta pwd - para mostrar la ruta open - para abrir finder mv - para mover archivos
