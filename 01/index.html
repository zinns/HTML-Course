HTML5 es la versión más actual. Cuando se crea un archivo nuevo siempre debe tener una extensión .html por ejemplo "index.html" ¿Cuáles son los comandos más utilizados de la terminal? mkdir, touch, ls, pwd, open, mv, entre otros. Ejemplos y descripción de comandos: mkdir - para crear una carpeta touch - para crear un archivo ls - para ver lo que hay dentro de una carpeta pwd - para mostrar la ruta open - para abrir finder mv - para mover archivos ¿Qué es git? Es un manejador o controlador de versiones. ¿Para qué sirve? Para comparar los cambios que se han hecho con versiones anteriores. ¿Cuáles son los comandos más utilizados? git add, git commit -m, git branch -v, git push -u origin, git init, git fetch, entre otros... Poner ejemplo y descripción de los comandos. git add . - para agregar los archivos que han tenido cambios git commit -m "___" - para agregar un commit git branch -v muestra el nombre de la rama y el último commit git push -u origin "nombre de la rama" - para enviar o subir cambios a github git init - para iniciar un repositorio en git git fetch - actualiza si hubo una nueva rama o versión de github ¿Qué es GitHub? Es un servidor en la nube que te permite trabajar en conjunto con otros desarrolladores y te permite hacer respaldos de la información. ¿Para qué sirve? Se utiliza principalmente para almecenar proyectos en la nube y que otros miembros puedan también acceder a la información, bajarla, hacer modificaciones y luego subir su parte actualizada para que todos dispongan de esta información. ¿Qué es un PR? Una Pull request funciona para integrar tus cambios con los de otro desarrollador en el proyecto principal. Estos cambios deben ser aprobados por otro colaborador del mismo proyecto. ¿Qué es un merge? Permite integrar otras ramas en una sola rama o en la rama principal.
