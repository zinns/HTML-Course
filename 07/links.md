# Instrucciones

1. Crear la estructura básica de HTML dentro del archivo `index.html`.

2. Dentro del archivo `html` recrear una tienda en línea (Apple, Zara, Forever21) usando enlances hacia esa página.

3. El contenido debe estar estructurado, incluir etiquetas de contenedores, texto, listas, formularios e imagen.

> ejemplo: [Video](./assets/example/example.mov)

# Recordatorio

- Debe salir una rama de `develop-[nombre]`, como se ha hecho en ejercicios anteriores.

- Nombre de las ramas siempre debe cumplir con las convensiones.
